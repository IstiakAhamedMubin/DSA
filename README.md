## 🧠 Key Concepts to Remember
📌 Time Complexity  — How fast does the algorithm run?
📌 Space Complexity — How much memory does it use?
Big O Notation:
O(1)       → Constant   — Best
O(log n)   → Logarithmic
O(n)       → Linear
O(n log n) → Log Linear
O(n²)      → Quadratic  — Avoid if possible
Golden Rules:
✔ Brute force first, then optimize
✔ Always think about edge cases
✔ Dry run your code before submitting
✔ Understand, don't memorize
